# Sundown Studio - TwoGoodCo Clone

A beautiful, modern website clone inspired by Sundown Studio / TwoGoodCo design. This is a multi-disciplinary studio website focused on creating unique, end-to-end experiences and environments.

## Live Demo

**[View Live Website](https://two-good-co-lac.vercel.app/)**

## Screenshots

### Hero Section
![Hero Section](https://two-good-co-lac.vercel.app/)

The website features:
- Stunning hero section with "SPACES THAT INSPIRE" typography
- Animated orange gradient background elements
- Clean navigation with Work, Studio, and Contact links
- Video background showcasing design work

### Features

- **Smooth Animations**: GSAP-powered scroll animations
- **Video Integration**: Embedded video showcasing design work
- **Interactive Project List**: Hover effects reveal project images
- **Modern Typography**: Custom fonts (Neue Haas Display)
- **Scrolling Marquee**: Dynamic text animation
